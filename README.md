# Computer Graphics: The Dummy Is You

A WebGL-based 3D graphics application with real-time pose detection using MediaPipe. Control and interact with 3D scenes using your webcam and body movements.

# Authors

- Jack Adkins
- Nate Pfeffer

## Features

- WebGL rendering with custom shaders
- Real-time pose detection and tracking
- Interactive 3D scene manipulation
- XML-based scene loading

## Running

Start a local server:
```
python -m http.server 8004
```

Then open `http://localhost:8004` in your browser.
